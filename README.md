**Steps to run the application in your local machine**

1. Install an Oracle database and create a schema
2. Go to src/main/resources/db and run pms.sql script
3. Change DB and SMTP config in application-dev.properties
4. Run mvn clean install to build the application
5. Run mvn spring-boot:run to start the application
6. Troubleshoot on need basis
